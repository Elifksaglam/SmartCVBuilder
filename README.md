****Smart CV Builder****
I created the "Smart CV Builder" to bridge the gap between simple text entries and professional design. 
This project was a 'learning by doing' journey where I focused on using Python functions to manage data flow.
**Why I built this:**
**Inspiration:** I wanted to replicate the ease of Canva in a local desktop application.

**Growth:** I used AI tools as a mentor to deepen my knowledge of advanced engineering concepts and to optimize my code.

**Access:** Instead of relying on paid memberships, I built my own tool to help others generate CVs for personal reasons
